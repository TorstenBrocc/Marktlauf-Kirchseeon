# Project Brief: ATSV Kirchseeon Marktlauf Homepage

## Overview
The goal is to provide a modern, functional, and mobile-friendly homepage for the "ATSV Kirchseeon Marktlauf" event.

- **URL:** http://www.atsv-kirchseeon-marktlauf.de/
- **Target Audience:** Runners (Bambini, Youth, Elite), spectators, and sponsors.
- **Primary Objective:** Provide event information, display routes, facilitate registration, and provide contact options.

## Design & Architecture Decisions
- **Color Palette:** 
    - Primary: `#009640` (ATSV Green)
    - Accent: `#ff6b35`
- **Layout:** Responsive single-page application (SPA) style with sticky navigation.
- **Key Sections:** Hero, Run Distances, Schedule (Timeline), Start & Finish (Map + GPX), Registration, Sponsors, and Contact.
- **Architecture:** Modular CSS (base, layout, components) for scalability.
